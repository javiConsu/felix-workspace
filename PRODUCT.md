## Summary

SalesHackersGTM is a SaaS platform powered by an AI agent that automates the entire GTM workflow for B2B sales teams in Spanish. It is the first GTM agent built natively for the Hispanic B2B market.

Key differentiators:
- Spanish-first — the only GTM agent built natively in Spanish.
- Integrated stack — Apollo, Instantly, Walead, HubSpot, Google in one agent.
- Autonomous signals — detects buying intent every morning without manual input.
- Learns from results — improves outreach based on what actually converts.
- Data privacy — deployed on the client infrastructure, data never leaves their server.
- Client-provided API keys — the product charges only for the intelligence layer.

See the full specification document for: onboarding flow, skill catalog design, credit system, freemium model, admin panel, feedback-for-credits mechanic, Apify subsidy strategy, tool abstraction layer, learning system, and all integration details.
